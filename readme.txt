This is Jquery based image horizontal slider plugin, it is using Tiny Carousel light weight jquery script to the slideshow. This is similar to my Tiny carousel horizontal slider plugin but it contains option to add Link, Alt Text to the images. In the plugin admin, you have options to add image URL, Link, Alt text.

= Plugin Features =

*   Easy to customize.
*   Support all browser.
*   Automatically pauses on mouse over.
*   Short code available for pages and posts.
*   Supports navigation by button.
*   Option to set a interval for automatic slide.
*   Supports localization
*   Responsive
